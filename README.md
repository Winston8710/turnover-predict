# turnover-predict
turnover predict
## 01. 系統開發目的
-離職率的上漲將導致公司內部的人才流失。
-為了改善公司離職率上漲的問題，我們開發離職預測系統並標示出對公司有益的人才，以協助主管及HR即時了解及慰留。

## 02. 預期效益
-快速了解員工狀態與離職預警，即時調配與掌握人才
